Privacy Policy

  • coinosis currently does not store any user data in its own servers.
  • coinosis does not run advertisements nor allows any other party to do so.
  • coinosis does not collect information from user's devices via cookies or any other means.
  • coinosis does access user data from the Ethereum blockchain and 3Box.

Ethereum blockchain

  • coinosis accesses the following user information from the Ethereum blockchain: wallet address.

3Box

  • coinosis accesses the following user information from 3Box: username and email.
  • coinosis stores the following user information in 3Box: username and preferred language.
  • In case a user does not have a 3Box profile, coinosis creates one for them.

YouTube

  • coinosis uses YouTube API Services. Please read the Google Privacy Policy .
  • coinosis does not access, collect, store or use any user information obtained from YouTube API Services.
  • coinosis does not access or use any Authorized Data from any YouTube account, except from its own.
